# Task Organizer
Web application for posting tasks and having them auto organize

## For the VueJS Client
[VueJS Client Repo](https://github.com/Nintron/TaskOrganizerClient)

## Initial project setup
```
npm install
```

### Run using nodemon for hot-reloading development
```
npm run dev
```

### Run using Node
```
npm run start
```

### For production
Set env variable "NODE_ENV" to "production", this will make express
use the public static folder of the compiled VueJS client
